# Affinity Unified App — November 2025 Major Update

## What Changed

Serif released "Affinity" as a single unified application in November 2025, replacing the three
separate apps (Affinity Photo 2, Affinity Designer 2, Affinity Publisher 2).

### Old → New Terminology

| Old (Affinity Photo 2) | New (Affinity unified) |
|------------------------|------------------------|
| Photo Persona          | Pixel Studio           |
| Liquify Persona        | Liquify tools (in Pixel Studio) |
| Develop Persona        | Develop tools (in Pixel Studio) |
| Export Persona         | Export (in Studio File menu) |
| Switch to Designer app | Switch to Vector Studio |
| Switch to Publisher app| Switch to Layout Studio |

### Pricing

- **Free**: Full professional toolset. Requires a free Canva account to activate.
- **Canva Pro** (~$14.99/month): Unlocks all AI "Magic" tools (Generative Fill, Magic Expand,
  Background Remover, Smart Vectorizer) and access to Canva's asset library and brand kits.
- Previous one-time purchase licenses are honored; existing users get the free tier automatically.

### UI Changes

- **Universal Toolbar**: Adapts dynamically to the active Studio — tools change based on context
- **Unified Layers Panel**: Handles vector, pixel, and text layers with equal priority;
  includes new filtering and search options
- **Studio Switcher**: Single control to move between Pixel / Vector / Layout Studios
- **Workspace Presets**: Save and recall custom panel/tool layouts per task or Studio
- **Canva Cloud sync**: Automatic versioning and cross-device access (local saving still works)

### New Capabilities

- **Unified file format**: One file works across all Studios; no cross-app export/import needed
- **Export to Canva**: Send assets directly into Canva as editable templates
- **Vector Warp Tool**: Non-destructive mesh warping of vector objects (Vector Studio)
- **ePub Export**: Reflowable eBook creation toolset (Layout Studio)

### What Did NOT Change

- Core non-destructive workflow (adjustment layers, live filters, masks) is unchanged
- Keyboard shortcuts for pixel editing tools are unchanged
- Blend modes, layer stack behavior, and clipping mask behavior are unchanged
- RAW development workflow is unchanged (now in Develop tools within Pixel Studio)
