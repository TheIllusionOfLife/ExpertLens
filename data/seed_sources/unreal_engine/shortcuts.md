# Unreal Engine 5 Essential Shortcuts

## Viewport Navigation
- Right-click+WASD: Fly mode | Right-click drag: Look around
- F: Focus on selected actor
- Alt+LMB: Orbit | Alt+RMB: Dolly zoom
- G: Toggle game view (hides editor widgets)
- P: Show/hide navigation mesh
- ` (backtick): Show all stats

## Transform
- W: Move gizmo | E: Rotate gizmo | R: Scale gizmo
- Space: Cycle through gizmo types
- End key: Snap to floor
- Ctrl+drag: Snap to grid
- Alt+drag: Duplicate while dragging

## Object & World
- Ctrl+D: Duplicate selected | Delete: Remove selected
- Ctrl+Z: Undo | Ctrl+Y: Redo
- Ctrl+S: Save current level
- Ctrl+A: Select all | Escape: Deselect all
- Ctrl+click: Add to selection
- L: Toggle visibility of selected actor

## Play & Simulate
- Alt+P: Play in viewport | Alt+S: Simulate
- F8: Eject from possessed pawn (while playing)
- Escape: Stop play/simulate
- Ctrl+P: Pause simulation

## Blueprint Editor
- Right-click empty space: Context menu for adding nodes
- Drag from pin: Create connected node
- Alt+click on wire: Remove connection
- Ctrl+W: Duplicate node
- C: Add comment box around selected nodes
- Compile: Ctrl+Alt+F7 (or click Compile button)
- B (in Blueprint): Add Branch node | S: Add Sequence

## Content Browser
- Ctrl+S: Save all | Ctrl+N: New asset
- Right-click: Context menu for import/create
- Double-click: Open asset editor
