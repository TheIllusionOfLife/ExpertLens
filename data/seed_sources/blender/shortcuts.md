# Blender Essential Shortcuts

## Navigation
- Middle Mouse: Orbit | Shift+Middle Mouse: Pan | Scroll: Zoom
- Numpad 0: Camera view | 1: Front | 3: Right | 7: Top
- Numpad 5: Toggle orthographic/perspective
- Ctrl+Alt+0: Align camera to current view
- `/` (Numpad): Local view (isolate selected)

## Object Mode
- G: Grab/Move | R: Rotate | S: Scale
- G/R/S + X/Y/Z: Constrain to axis
- G/R/S + Shift+X/Y/Z: Constrain to plane
- A: Select All | Alt+A: Deselect All
- Shift+D: Duplicate | Alt+D: Linked Duplicate
- Ctrl+D: (not standard — use Shift+D)
- X or Delete: Delete | M: Move to collection
- Ctrl+J: Join objects | P: Separate
- Ctrl+A: Apply transforms (MUST do before rigging)
- Tab: Toggle Edit Mode | Ctrl+Tab: Pie menu for modes

## Edit Mode
- 1/2/3: Vertex/Edge/Face select mode
- Alt+click: Select edge/face loop
- Ctrl+click: Select shortest path
- E: Extrude | Alt+E: Extrude menu
- Ctrl+R: Loop Cut (scroll to add cuts, right-click confirms)
- K: Knife tool (Enter to confirm)
- I: Inset faces
- Ctrl+B: Bevel | B: Box select
- O: Proportional editing toggle
- Shift+N: Recalculate normals
- M: Merge | Alt+M: Merge (older versions)
- F: Fill (edge/face)
- J: Connect vertices with edge

## Modifiers & Properties
- Ctrl+1/2/3: Subdivision Surface modifier quick-add
- Ctrl+M: Mirror modifier setup
- Tab in Properties: Switch between panels

## Animation (CHANGED in Blender 4.0)
- **I: Insert keyframe immediately** (4.0+) — no longer opens a menu
- **K: Keyframe type picker menu** (4.0+) — this is what I used to do in 3.x
- Ctrl+I: Insert keyframe with custom type dialog

## Rendering
- F12: Render image | Ctrl+F12: Render animation
- F11: Show last render
- Z: Shading pie menu (Solid/Material/Rendered/Wireframe)
