Technical Challenges Solved

2-minute image session limit → SlidingWindow compression
Sessions run 30+ minutes with continuous screen sharing
~10-min WebSocket timeout → sessionResumption
Handle-based reconnect in ~2 seconds, full context preserved
RAG latency (200-500ms) → Context stuffing (0ms)
50-70 pages of curated knowledge loaded into system instruction at session start
Tool calls pause audio → NON_BLOCKING + WHEN_IDLE
Tools execute between turns, never interrupting the voice stream
